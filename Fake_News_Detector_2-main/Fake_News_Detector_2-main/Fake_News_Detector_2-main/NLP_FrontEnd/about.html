Project Objective

This project aims to detect fake news by leveraging a Retrieval-Augmented Generation (RAG) approach combining real-time evidence from multiple sources with the reasoning power of Large Language Models (LLMs). Our system enhances reliability, transparency, and up-to-date fact-checking capability.

Project Flow

The diagram below outlines the core flow from user input to verdict delivery.

User Input
Claim Extraction
Multi-source Evidence Retrieval
LLM Agentic Prompting
Model Inference
Verdict: FAKE / REAL / UNCERTAIN

Further Improvements

  • Improve claim extraction using advanced NLP Named Entity Recognition (NER) to better target evidence search.
  • Expand evidence sources by integrating more news APIs, trusted fact-checker RSS feeds, and social media signals.
  • Implement local vector search for efficient retrieval from large cached corpora.
  • Add multimodal inputs such as images or videos for richer context.
  • Enhance UI/UX with real-time search suggestions and progressive loading states.
  • Document API usage and backend architecture in a dedicated wiki for maintainability.
